JABCrypt

JABCrypt is a File integrity Manager which ensure that all the files stored in the system is secure.

  • Project Type: Academic Minor Project
  • Field: Cybersecurity
  • Completed: 2 May 2023

JABCrypt was developed to ensure that the files stored in the system is secure and has not been tampered with by an external malicious entity. We use hashing functions to convert the files into hash and store it in a specific file. The hashes stored in this file is compared with the current hash of the file for any discrepancies. If any are found, that entails that the file has been compromised and it can be reverted to the last checkpointed state to when it was not tampered by an external entity.

VoiceMate

A Voice Controlled ChessBoard for the Visually Impaired

  • Project Type:: Academic Major Project
  • Field: Assistive Technology
  • Completed: 10 May 2024

The VoiceMate chessboard allows Visually Impaired individuals to independantly play chess by issuing voice commands for moving the chess pieces on the board, eliminating the need for an intermediary assistance. Some of the features of the chessboard are an inbuilt AI to play against, Possible Movement are shown via the software interface and Incorrect moves are not allowed. The ChessBoard was built from old parts of electronic device to increase sustainability and reduce costs. Some of the hardware technologies used were Stepper Motors, TB6600 Driver Module, Electromagnet, Arduino Uno Atmega 328. The software technologies used were Python and the Arduino IDE.
